# AnimaFelis Mood Logger 

An Android mood tracking app- the first prototype for my Final Year Project, AnimaFelis.

## About
AnimaFelis is a pixel-art emotional virtual pet mobile game where players raise
a cat spirit's egg through daily emotional interactions. 
This Mood Logger is the core mechanic prototype, built to validate the emotional check-in system before
full game development.

## Features
- Select a mood from 5 emoji states (Happy, Sad, Tired, Anxious, Calm)
- Write a short daily reflection
- View mood history with timestamps in a scrollable log

## Tech Stack
- Platform: Android
- Language: Java
- Min SDK: API 24 (Android 7.0)
